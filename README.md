# Book Review App

A simple Node.js + Express app that lets users create, view, edit, and delete book reviews.

## Features

- Add new reviews
- Edit existing reviews
- Delete reviews
- View all reviews

## Technologies

- Node.js
- Express
- Pug (template engine)
- CSS

## How to Run

1. Install dependencies: npm install
2. Start the app: npm run dev
3. Visit http://localhost:3000
